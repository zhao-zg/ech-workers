v1.0.8 - 版本检查与更新功能

## 新功能

### 全平台版本检查
- **命令行工具**: 添加 -version 和 -check-update 参数
  - ech-workers -version: 显示当前版本号
  - ech-workers -check-update: 手动检查更新
  - 启动时自动在后台检查更新(延迟2秒,不影响启动速度)

- **Android 客户端**: 设置界面版本管理
  - 显示当前版本号
  - 一键检查更新按钮
  - 更新对话框显示版本对比和 Release Notes
  - 支持直接跳转到 GitHub 下载页面

- **OpenWrt 客户端**: LuCI 界面版本信息
  - 自动显示当前版本
  - 检查更新按钮
  - 更新结果页面显示详细对比和更新内容
  - 直接链接到 GitHub Release 页面

## 技术实现

### 版本检查核心
- **GitHub Release API**: 自动获取最新版本信息
- **版本比较算法**: 智能比较语义化版本号
- **错误处理**: 网络失败时静默处理,不影响程序运行
- **超时控制**: 10秒超时,避免长时间等待

### 代码改进
- 	unnel/version.go: 版本检查和比较核心逻辑
- ndroid/VersionChecker.java: Android 异步更新检查
- openwrt/ech-workers.lua: LuCI 版本显示集成
- README.md: 更新文档,添加版本检查说明

## 用户体验优化

### 自动更新提示
- 程序启动2秒后自动检查更新
- 发现新版本时友好提示
- 包含版本号、发布时间、更新内容
- 检查失败不影响正常使用

### 手动检查
- 命令行: 即时反馈检查结果
- Android: 进度对话框显示检查状态
- OpenWrt: 专用更新结果页面
